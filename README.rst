Install
------------------------------------------------------------------------------

``aws_glue_artifact`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install aws-glue-artifact

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade aws-glue-artifact
